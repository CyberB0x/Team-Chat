# 💬 Real-time Chat App (Django + React + WebSockets + JWT)

## 🚀 Features
- 🔐 JWT Authentication (Login / Register)
- 💬 Real-time Messaging via Django Channels
- 🧩 React Frontend with Tailwind CSS
- 📡 WebSocket Connection per Room
- 👥 Multiple Chat Rooms
- 🧭 Protected Routes + Auto Login Redirect
- 🎨 Responsive UI

## 🛠️ Tech Stack
- Backend: Django, Django REST Framework, Channels, Redis
- Frontend: React, Axios, Tailwind CSS
- Auth: SimpleJWT
- DB: SQLite / PostgreSQL

## 🧩 Run Locally

### Backend
```bash
  cd backend
  python manage.py runserver
```

## 📦 Build for Production
```commandline
   npm run build
```
